Lightning — 快速页面生成解决方案。
=========


特性：
----
- 适用于简单的静态页面生成（如仅有区块超链接功能的运营页面）。
- 通过选区遮罩超链接方式快速生成页面。
- 选区可定制大小以及旋转定位，甚至为不规则形状。
- 生成的页面自动适应屏幕。
- 可定制模板。
